Notification Bell Integration Demo

This demo shows how to integrate the notification count endpoints with a notification bell icon.

API Endpoints Used:

  • GET /api/notification/unread-count/{userId} - Get unread notification count
  • GET /api/notification/stats/{userId} - Get notification statistics
  • GET /api/notification/all_userNotifications/{userId} - Get all notifications
  • PUT /api/notification/markAsRead/{notificationId} - Mark notification as read
